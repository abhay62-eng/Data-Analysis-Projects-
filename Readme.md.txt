# Zomato Restaurant Analysis

## Project Overview
This project analyzes Zomato restaurant data to discover insights about restaurant ratings, cuisines, pricing, online delivery, and customer preferences.

The analysis was performed using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn.

---

## Objectives

- Analyze restaurant ratings
- Find popular cuisines
- Compare online delivery vs offline restaurants
- Study pricing trends
- Visualize restaurant distribution

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset Information

The dataset contains information about:
- Restaurant names
- Locations
- Ratings
- Votes
- Average cost
- Cuisines
- Online delivery availability

---

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insights & Conclusions

---

## Key Insights

- Most restaurants offer online delivery
- North Indian cuisine is highly popular
- Higher-rated restaurants generally receive more votes
- Certain locations have significantly more restaurant density
